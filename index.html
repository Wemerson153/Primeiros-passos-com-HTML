História do Computador

A palavra "computador" vem do verbo "computar" que significa "calcular". Sendo assim, conseguimos pensar que a criação dos computadores começa na idade antiga, já que a relação de contar já intrigava os homens.

Breve linha do tempo dos computadores

  • VIII A.C - Abáco
  • 1642 - Máquina de Pascal
  • 1671 - Máquina de calcular de leibnitz
  • 1820 - Charles Babbage: "Computador Analítico"
  • 1886 - Herman Hollerit: cartão perfurado
  • 1936 - Primeiro computador: Z-1
  • 1940-1955 - Primeira geração de computadores: válvulas
  • 1944 - MARK I
  • 1945 - ENIAC
  • 1955-1965 - Segunda geração de computadore: transístores
  • 1965-1980 - Terceira geração de computadores:circuitos integrados
  • 1980-1990 - Quarta geração de computadores: circuitos de larga escala
  • 1990-até hoje - Quinta geração de computadores: Ultra Large Scale Integration

Esses são apenas alguns marcos importantes na história dos computadores. Desde os primeiros dispositivos de cálculo até os avanços tecnológicos mais recentes, os computadores têm desempenhado um papel fundamental no progresso humano e na transformação de várias indústrias.

O que são Clients e Servers?

Um Client é um dispositivo, como um computadorm smartphone ou tablet, que solicita e consome serviços ou recursos de um servidor. Os clientes são responsáveis por enviar solicitações ao servidor e receber as respostas correspondentes. Eles normalmente executam aplicativos ou softwares que interagem com o servidor para obter informações, acessar arquivos, enviar mensagens, entre outras tarefas.

Um Servidor é um computador ou sistema que responde às solicitações dos clientes. Ele é projetado para fornecer serviços, recursos ou informações específicas aos clientes que o solicitam. Os servidores geralmente tem maior capacidade de processamento, armazenamento e conexão de rede do que os clientes, permitindo que atendam a várias solicitações de clientes simultaneamente.

Entendendo a diferença entre listas ordenadas e não ordenadas

As listas ordenadas e não ordenadas são dois tipos diferentes de listas que podem ser criadas em HTML.
Uma lista ordenada, representada pela tag "< ol >", é utilizada quando a ordem dos itens na lista é importante. Os itens são exibidos sequencialmente e cada item é precedido por um número ou letra. Por exemplo, uma lista ordenada pode ser usada para enumerar passos de um tutorial ou para listar itens em uma hierarquia. Os itens são marcados com tags "< li >" (item de lista). O navegador atribui automaticamente os números ou letras a cada item da lista. Por outro lado, uma lista não ordenada, representada pela tag "< ul >", é utilizada quando a ordem dos itens não é relevante. Os itens são exibidos como uma lista de itens com marcadores, geralmente pontos. Uma lista não ordenada é útil quando a sequência dos itens não importa, como uma lista de recursos ou características. Os itens também são marcados com tags "< li >".
Ambos os tipos de listas, ordenadas e não ordenadas, permitem criar estruturas organizadas de informações em uma página HTML. A escolha entre usar uma lista ordenada ou não ordenada depende da natureza dos dados que estão sendo apresentados e se a ordem dos itens é relevante ou não
